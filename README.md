# dotfiles
my dotfiles for 
- git/git bash
- ...

> [!IMPORTANT]
> Codespaces で XDG 設定を使う場合
> 
> Codespaces では ~/.config が既に存在するため、
> dotfiles の .config ディレクトリは自動でリンクされないことがある。
> そのため install.sh で個別にシンボリックリンクを作成する。
> （スクリプトの実行は Codespaces 側で[やってくれる](https://docs.github.com/en/codespaces/setting-your-user-preferences/personalizing-github-codespaces-for-your-account#dotfiles)）
